# Przedmioty podręczne

Przedmioty podręczne zajmują miejsce przy pasku postaci. Nie są one zużywalne pernamentnie, zamiast tego odnawiają na początku każdej walki. Nie można korzystać z przedmiotów podręcznych, które nie są umieszczone przy pasku postaci.
